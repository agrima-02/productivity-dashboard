# 📘 Productivity Dashboard

A modern, interactive productivity dashboard built using HTML, CSS, and JavaScript, designed to help users efficiently manage tasks, track daily habits, and monitor productivity.

# 🚀 Overview

This project is a fully client-side web application that combines task management with health tracking features. It focuses on simplicity, usability, and a visually appealing interface using a custom pink–purple theme.

# ✨ Features

	# 📝 Task Management
	
	* Add tasks with title, priority, and due date
	* Drag and drop tasks across:
	
	  * To Do
	  * In Progress
	  * Done
	* Edit tasks directly from the dashboard
	* Delete tasks
	* Search and filter tasks by priority
	* Sort tasks by due date
	* Highlight:
	
	  * Overdue tasks (red)
	  * Tasks due soon (orange)
	
	# ⭐ Pinned Tasks
	
	* Pin important tasks
	* Dedicated pinned section at the top
	* Automatically appears only when needed
	* Unpin option available
	* Displays tasks in a responsive grid layout
	
	# 📊 Progress Tracking
	
	* Shows total and completed tasks
	* Dynamic progress bar updates in real time
	
	# 💧 Health Tracker
	
	* Water tracker:
	
	  * Add or remove intake
	  * View recent logs
	* Steps tracker:
	
	  * Log daily steps
	  * View history
	* Data persists using localStorage
	
	# 🎯 Daily Goals
	
	* Tracks:
	
	  * Water goal (8 glasses)
	  * Steps goal (10,000 steps)
	* Real-time updates based on user activity
	
	# 🔐 Authentication
	
	* Login system with two predefined users
	* Remember Me functionality
	* Session handling
	* Invalid login message
	* Logout option
	
	# 🎨 UI & Experience
	
	* Clean and minimal pink–purple themed UI
	* Dark mode support
	* Multiple themes:
	
	  * Default (Purple)
	  * Blue
	  * Yellow
	  * Green
	* Smooth hover and transition effects
	* Responsive layout for smaller screens

# 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)
* LocalStorage API
